# Customer Segmentation and Recommendation System

## Project Overview

This project uses Machine Learning and Clustering techniques to segment customers based on their characteristics and provide recommendations for each customer group.

The project was built as part of my Data Science internship preparation to gain hands-on experience in data analysis, clustering, and predictive modeling.

---

## Dataset

- Dataset: Mall Customers Dataset
- Features:
  - Gender
  - Age
  - Annual Income (k$)
  - Spending Score (1-100)

---

## Objectives

- Perform Exploratory Data Analysis (EDA)
- Segment customers using K-Means Clustering
- Analyze customer groups
- Train a prediction model to predict cluster labels
- Provide recommendations based on customer segments
